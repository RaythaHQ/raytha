Raytha-Specific Filters

raytha_attachment_url

Get the public URL for an attachment:

<img src="{{ Target.PublishedContent.image | raytha_attachment_url }}" alt="...">

get_navigation_menu

Retrieve a navigation menu by developer name:

{% assign footer = 'footer_menu' | get_navigation_menu %}
{% for item in footer.Items %}
  <a href="{{ item.Url }}">{{ item.Label }}</a>
{% endfor %}
