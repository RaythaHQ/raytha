Liquid Template Basics

Raytha uses the Liquid templating language to render dynamic content. Learn the fundamentals of Liquid syntax.

What is Liquid?

Liquid is a template language created by Shopify. It's safe, customer-facing, and easy to learn. Raytha uses Liquid for:

  • Web templates (page layouts)
  • Widget templates (Site Page widgets)
  • Email templates

Basic Syntax

Liquid has three main components:

Objects (Output)

Output data using double curly braces:

{{ Target.Title }}
{{ CurrentOrganization.OrganizationName }}

Tags (Logic)

Control flow and logic using curly braces with percent signs:

{% if Target.IsPublished %}
  <span>Published</span>
{% endif %}

Filters

Modify output using the pipe character:

{{ Target.Title | upcase }}
{{ Target.CreationTime | date: "%B %d, %Y" }}

Conditional Logic

if / elsif / else

{% if Target.IsPublished %}
  <span class="badge bg-success">Published</span>
{% elsif Target.IsDraft %}
  <span class="badge bg-warning">Draft</span>
{% else %}
  <span class="badge bg-secondary">Unknown</span>
{% endif %}

unless

{% unless Target.IsPublished %}
  <span>Not yet published</span>
{% endunless %}

case / when

{% case Target.Status %}
  {% when 'draft' %}
    <span>In Progress</span>
  {% when 'published' %}
    <span>Live</span>
  {% else %}
    <span>Unknown</span>
{% endcase %}

Operators

Operator Description
==Equals
!=Not equals
>Greater than
<Less than
>=Greater than or equal
<=Less than or equal
orLogical or
andLogical and
containsString/array contains value

Loops

for Loop

{% for item in Target.Items %}
  <div>{{ item.PrimaryField }}</div>
{% endfor %}

Loop Variables

{% for item in Target.Items %}
  {{ forloop.index }}      // 1, 2, 3...
  {{ forloop.index0 }}     // 0, 1, 2...
  {{ forloop.first }}      // true on first iteration
  {{ forloop.last }}       // true on last iteration
  {{ forloop.length }}     // total items
{% endfor %}

Loop with limit and offset

{% for item in Target.Items limit:5 %}
  // First 5 items
{% endfor %}

{% for item in Target.Items offset:2 %}
  // Skip first 2 items
{% endfor %}

else in Loops

{% for item in Target.Items %}
  <div>{{ item.PrimaryField }}</div>
{% else %}
  <p>No items found.</p>
{% endfor %}

Variables

assign

{% assign page_title = Target.Title %}
<h1>{{ page_title }}</h1>

capture

{% capture full_name %}{{ Target.FirstName }} {{ Target.LastName }}{% endcapture %}
<p>Welcome, {{ full_name }}</p>

Common Filters

String Filters

{{ "hello" | upcase }}          // HELLO
{{ "HELLO" | downcase }}        // hello
{{ "hello" | capitalize }}      // Hello
{{ "hello world" | truncate: 8 }} // hello...
{{ "  hello  " | strip }}       // hello
{{ "hello" | append: " world" }} // hello world
{{ "hello" | prepend: "say " }} // say hello
{{ "hello" | replace: "e", "a" }} // hallo
{{ "hello-world" | split: "-" }} // ["hello", "world"]

HTML Filters

{{ Target.Content | escape }}    // HTML escape
{{ Target.Content | strip_html }} // Remove HTML tags
{{ Target.Url | url_encode }}    // URL encode

Number Filters

{{ 4.5 | round }}     // 5
{{ 4.5 | ceil }}      // 5
{{ 4.5 | floor }}     // 4
{{ 1000 | plus: 5 }}  // 1005
{{ 10 | minus: 3 }}   // 7
{{ 5 | times: 2 }}    // 10
{{ 10 | divided_by: 3 }} // 3

Array Filters

{{ Target.Items | size }}         // Count items
{{ Target.Items | first }}        // First item
{{ Target.Items | last }}         // Last item
{{ Target.Items | join: ", " }}   // Join with comma
{{ Target.Items | sort: "Title" }} // Sort by property
{{ Target.Items | reverse }}      // Reverse order

Date Filters

{{ Target.CreationTime | date: "%B %d, %Y" }}  // January 15, 2024
{{ Target.CreationTime | date: "%Y-%m-%d" }}   // 2024-01-15
{{ "now" | date: "%H:%M" }}                    // Current time

Default Values

{{ Target.Subtitle | default: "No subtitle" }}

Checking for Blank/Empty

{% if Target.Description != blank %}
  {{ Target.Description }}
{% endif %}

{% if Target.Items.size > 0 %}
  // Has items
{% endif %}

Comments

{% comment %}
  This won't be rendered in the output
{% endcomment %}

Raw Output

Output Liquid code without processing:

{% raw %}
  {{ this will not be processed }}
{% endraw %}

Template Inheritance

Inherit from a base layout:

{% layout "raytha_html_base_layout" %}

In the parent template, render child content:

{{ content_for_layout }}
