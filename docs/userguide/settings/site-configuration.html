Site Configuration

Configure your website's basic settings including organization name, URL, timezone, and homepage.

Accessing Configuration

Navigate to Settings > Configuration.

Organization Settings

Organization Name

Your organization or website name. Displayed in:

  • Admin panel header
  • Email templates (via {{ CurrentOrganization.OrganizationName }})
  • Public templates

Website URL

The public URL of your website (e.g., https://example.com). Used for:

  • Generating absolute URLs in emails
  • Redirects and canonical URLs

Date & Time Settings

Timezone

Select your timezone from the dropdown. Affects:

  • Date display in admin panel
  • Scheduled content (if applicable)
  • Timestamps in templates

Date Format

Choose how dates are displayed throughout the system.

Homepage Settings

Configure what displays when visitors access your root URL (/).

Homepage Type

  • Site Page: Use a Site Page as your homepage
  • Content Item: Use a specific content item
  • View: Use a content type view (list page)

Selecting the Homepage

  1. Choose the Home Page Type
  2. Select the specific page, item, or view from the dropdown
  3. Click Save

Active Theme

Select which theme renders your public website. Only published themes appear in the dropdown.

  1. Select a theme from Active Theme
  2. Click Save
Note: Changing the active theme immediately affects your public website. Test new themes thoroughly before switching.

Saving Changes

Click Save at the bottom of the page. Changes take effect immediately.

Configuration in Templates

Access these settings in your Liquid templates:

{{ CurrentOrganization.OrganizationName }}
{{ CurrentOrganization.WebsiteUrl }}
{{ CurrentOrganization.TimeZone }}
{{ CurrentOrganization.DateFormat }}
