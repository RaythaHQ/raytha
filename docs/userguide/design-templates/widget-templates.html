Widget Templates

Widget templates control how individual widgets are rendered on Site Pages. Customize the HTML output for Hero, Card, CTA, and other widget types.

What are Widget Templates?

When you add a widget to a Site Page, the widget template determines how it's displayed. Each widget type has its own template that you can customize.

Accessing Widget Templates

  1. Go to Design > Themes
  2. Click on your active theme
  3. Click Widget Templates

Built-in Widget Types

Widget Developer Name Purpose
Hero hero Large banner sections with headline and CTA
WYSIWYG wysiwyg Rich text content blocks
Card card Bordered content cards with image and button
CTA cta Call-to-action sections
Content List content_list Dynamic lists of content items

Editing a Widget Template

  1. Click on a widget template
  2. Modify the Liquid code in the editor
  3. Click Save

Accessing Widget Settings

Widget settings are available through the Target object:

{{ Target.headline }}
{{ Target.content }}
{{ Target.buttonText }}
{{ Target.backgroundColor }}

Example: Hero Widget

<section style="background-color: {{ Target.backgroundColor | default: '#0d6efd' }}; 
                color: {{ Target.textColor | default: '#ffffff' }}; 
                min-height: {{ Target.minHeight | default: 400 }}px;">
  <div class="container py-5 text-{{ Target.alignment | default: 'center' }}">
    {% if Target.headline != blank %}
      <h1 class="display-4 fw-bold">{{ Target.headline }}</h1>
    {% endif %}
    
    {% if Target.subheadline != blank %}
      <p class="lead">{{ Target.subheadline }}</p>
    {% endif %}
    
    {% if Target.buttonText != blank %}
      <a href="{{ Target.buttonUrl }}" class="btn btn-{{ Target.buttonStyle }}">
        {{ Target.buttonText }}
      </a>
    {% endif %}
  </div>
</section>

Example: Content List Widget

The Content List widget includes a Target.items array:

<div class="row">
  {% for item in Target.items %}
    <div class="col-md-4">
      <div class="card">
        <div class="card-body">
          <h5>{{ item.PrimaryField }}</h5>
          <p>{{ item.PublishedContent.content | strip_html | truncate: 100 }}</p>
          <a href="/{{ item.RoutePath }}">Read more</a>
        </div>
      </div>
    </div>
  {% endfor %}
</div>

Common Settings by Widget

Hero

  • Target.headline, Target.subheadline
  • Target.backgroundColor, Target.textColor
  • Target.buttonText, Target.buttonUrl, Target.buttonStyle
  • Target.alignment, Target.minHeight

Card

  • Target.title, Target.description
  • Target.imageUrl
  • Target.buttonText, Target.buttonUrl

Content List

  • Target.headline, Target.subheadline
  • Target.contentType, Target.pageSize
  • Target.items (array of content items)

Best Practices

  • Use the default filter for fallback values
  • Check for blank values before rendering: {% if Target.headline != blank %}
  • Escape user content: {{ Target.title | escape }}
  • Test widgets with various content lengths

Revisions

Widget templates support revisions. Click Revisions to view or restore previous versions.
