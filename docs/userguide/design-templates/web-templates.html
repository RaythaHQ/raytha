Web Templates

Web templates control how your content is displayed on the public website. Use the Liquid templating language to create dynamic layouts.

What is a Web Template?

Web templates are HTML files with Liquid code that:

  • Define page layouts and structure
  • Display content from content items
  • Include navigation, headers, and footers
  • Handle lists and detail pages

Accessing Web Templates

  1. Go to Design > Themes
  2. Click on your active theme
  3. Click Web Templates

Template Types

Base Layouts

Parent templates that define the overall page structure (header, footer, navigation). Other templates inherit from these.

Page Templates

Templates for specific page types:

  • Content Item Detail: Displays a single content item
  • Content Item List: Displays a view/list of items
  • Home Page: Landing page template
  • Error Pages: 404, 403, 500 error pages
  • Site Page Templates: Templates for Site Pages with widget zones

Login Templates

Templates for authentication pages: login, registration, password reset.

Editing a Template

  1. Click on a template to open the editor
  2. Modify the HTML and Liquid code
  3. Click Save

Template Inheritance

Templates can inherit from a parent template using:

{% layout "raytha_html_base_layout" %}

The parent template uses {{ content_for_layout }} to render child content.

Basic Liquid Syntax

Output Variables

{{ Target.Title }}
{{ Target.PublishedContent.description }}

Conditionals

{% if Target.IsPublished %}
  <span class="badge bg-success">Published</span>
{% endif %}

Loops

{% for item in Target.Items %}
  <h2>{{ item.PrimaryField }}</h2>
{% endfor %}

Assigning Templates to Content

Templates must be assigned to content types or views:

  1. Go to the template's settings
  2. Under Access, select which content types can use this template
  3. Enable Allow for new content types to automatically include future types

Revisions

Each time you save a template, a revision is created. To restore a previous version:

  1. Open the template
  2. Click Revisions
  3. Select a version and click Restore

Creating a Template

  1. Click Create Template
  2. Enter:
    • Label: Display name
    • Developer Name: Unique identifier
    • Parent Template: Template to inherit from
  3. Add your HTML and Liquid code
  4. Click Create
Tip: See the Developer Guide for detailed Liquid template documentation and available variables.
