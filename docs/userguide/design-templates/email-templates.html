Email Templates

Customize the emails Raytha sends for password resets, magic link logins, and user notifications.

Accessing Email Templates

Navigate to Design > Email Templates to see all system emails.

Built-in Email Templates

Template Purpose
Admin Password Reset Sent when an admin requests a password reset
Admin Welcome Sent when a new admin account is created
User Password Reset Sent when a public user requests password reset
User Welcome Sent when a new public user registers
Magic Link Login Sent for passwordless login requests
Email Verification Sent to verify email addresses

Editing an Email Template

  1. Click on a template to open the editor
  2. Modify the following:
    • Subject: Email subject line (supports variables)
    • Content: HTML body of the email
    • CC: Optional carbon copy addresses
    • BCC: Optional blind carbon copy addresses
  3. Click Save

Available Variables

Use these Liquid variables in your email templates:

All Templates

{{ CurrentOrganization.OrganizationName }}
{{ CurrentOrganization.WebsiteUrl }}

User Templates

{{ Payload.FirstName }}
{{ Payload.LastName }}
{{ Payload.EmailAddress }}

Password Reset

{{ Payload.NewPassword }}
{{ Payload.LoginUrl }}

Magic Link

{{ Payload.MagicLinkUrl }}
{{ Payload.ExpiresAt }}

Inserting Variables

Click the Insert Variable button above the editor to see available variables and insert them at your cursor position.

Revisions

Email templates support revisions. Click Revisions to view or restore previous versions.

Testing Emails

To test email delivery:

  1. Ensure SMTP is configured in Settings > SMTP
  2. Trigger the email action (e.g., create a test user)
  3. Check the recipient's inbox
