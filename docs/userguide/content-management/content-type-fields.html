Content Type Fields

Fields define what data can be stored in each content item. Add fields to capture text, images, dates, and more.

Managing Fields

  1. Navigate to your content type
  2. Click Fields in the submenu
  3. Click Create Field to add a new field

Field Types

Type Description Use Case
Single Line Text Short text input Titles, names, short descriptions
Long Text Multi-line plain text Summaries, excerpts
WYSIWYG Rich text editor with formatting Article content, descriptions
Number Numeric values Prices, quantities, ratings
Date Date picker Event dates, publish dates
Checkbox True/false toggle Featured flag, active status
Dropdown Single selection from list Categories, status
Multiple Select Multiple selections from list Tags, features
Attachment File upload Images, documents, downloads
One-to-One Relation Link to single content item Author, category
One-to-Many Relation Link to multiple content items Related posts, gallery items

Creating a Field

  1. Click Create Field
  2. Enter the field details:
    • Label: Display name (e.g., "Featured Image")
    • Developer Name: Code identifier (e.g., "featured_image")
    • Field Type: Select from dropdown
    • Required: Whether the field must have a value
    • Description: Help text shown to editors
  3. Click Create

Dropdown and Multiple Select Options

For dropdown and multiple select fields, define the available choices:

  1. Enter each option on a new line
  2. Format: value|Display Label or just value
draft|Draft
published|Published
archived|Archived

Relationship Fields

Link content items together:

  • One-to-One: Select a single related item (e.g., Post → Author)
  • One-to-Many: Select multiple related items (e.g., Product → Related Products)

When creating a relationship field, select the Related Content Type to link to.

Field Order

Drag and drop fields in the list to reorder them. The order determines how fields appear in the content editor.

Deleting Fields

Warning: Deleting a field removes all data stored in that field across all content items. This cannot be undone.
