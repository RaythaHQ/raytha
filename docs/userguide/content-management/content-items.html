Content Items

Content items are the individual pieces of content within a content type. Create, edit, and publish your website's content here.

Viewing Content Items

Click any content type in the sidebar to see its items. Use the list view to browse, search, and filter content.

Creating a Content Item

  1. Navigate to the content type
  2. Click Create [Content Type Name]
  3. Fill in the fields
  4. Click Save as Draft or Publish

Draft vs Published

Content items have two states:

  • Draft: Work in progress, not visible on the public website
  • Published: Live and visible to website visitors

You can continue editing a published item. Changes are saved as a draft until you publish again.

Publishing Workflow

  • Save as Draft: Save changes without publishing
  • Publish: Make content live on your website
  • Unpublish: Remove from public website (keeps content)
  • Discard Draft: Revert to last published version

Editing Content

Click any content item to open the editor. The available fields depend on your content type configuration.

WYSIWYG Editor

Rich text fields include formatting tools:

  • Bold, italic, underline
  • Headings and paragraphs
  • Bullet and numbered lists
  • Links and images
  • Tables and code blocks

Attachments

For attachment fields:

  1. Click Choose File or drag and drop
  2. Select from existing media or upload new
  3. The file uploads to your media library automatically

Content Item Settings

Click Settings in the item submenu to configure:

  • URL Path: Custom URL for this item
  • Template: Which web template renders this item

Revisions

Raytha saves a revision each time you publish. To view or restore previous versions:

  1. Open the content item
  2. Click Revisions
  3. Browse the revision history
  4. Click Restore to revert to that version

Deleting Content Items

To delete a content item:

  1. Open the item or select it in the list
  2. Click Delete
  3. Confirm the deletion

Deleted items are moved to the trash and can be restored from Deleted Items in the content type menu.
