Initial Setup

Complete the first-time setup wizard to configure your Raytha installation and create your super admin account.

Accessing the Setup Wizard

When you first navigate to your Raytha installation, you'll be automatically redirected to the setup wizard if no configuration exists.

Step 1: Super Admin Account

Create your administrator account with the following information:

  • First Name and Last Name: Your display name in the admin panel
  • Email Address: Used for login and system notifications
  • Password: Must be at least 8 characters

Step 2: Organization Settings

Configure your organization's basic information:

  • Organization Name: Displayed in the admin panel header
  • Website URL: The public URL of your website (e.g., https://example.com)
  • Timezone: Used for displaying dates and scheduling

Step 3: Email Configuration

Set up email delivery for password resets and notifications:

  • SMTP Host: Your email server address
  • SMTP Port: Usually 587 for TLS or 465 for SSL
  • SMTP Username/Password: Credentials for authentication
  • Default From Address: The sender email address
  • Default From Name: The sender display name
Tip: If SMTP environment variables are configured on your server, you can skip entering SMTP settings here.

Completing Setup

Click Complete Setup to finish. Raytha will:

  1. Create your super admin account
  2. Set up the default theme and templates
  3. Create sample content (Posts content type with example articles)
  4. Configure the default navigation menu
  5. Set up the homepage

You'll be redirected to the login page where you can sign in with your new credentials.
