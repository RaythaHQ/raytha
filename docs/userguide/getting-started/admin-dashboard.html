Admin Dashboard

Learn how to navigate the Raytha admin panel and access key features.

Dashboard Overview

After logging in, you'll see the admin dashboard with quick access to common tasks and recent activity.

Main Navigation

The left sidebar provides access to all admin features:

Section Description
Dashboard Home screen with quick actions and overview
Content Types Manage content structures and their items
Site Pages Create standalone pages with widgets
Media Upload and manage files and images
Design Themes, web templates, and widget templates
Navigation Manage menus for your website
Users Manage public user accounts
Settings Configuration, admins, roles, and authentication

Top Navigation Bar

The top bar includes:

  • Organization Name: Click to return to dashboard
  • View Site: Opens your public website in a new tab
  • Profile Menu: Access your profile settings and logout

Content Type Quick Access

Your content types appear in the sidebar under their own section. Click any content type to view and manage its items.

Keyboard Shortcuts

Speed up your workflow with these shortcuts:

  • Ctrl + S: Save current form
  • Esc: Close modal dialogs

Your Profile

Click your name in the top-right corner to access:

  • Edit Profile: Update your name and email
  • Change Password: Update your login password
  • Logout: Sign out of the admin panel
