Site Pages

Site Pages let you create standalone pages with a flexible widget-based layout. Perfect for homepages, landing pages, about pages, and more.

What are Site Pages?

Unlike Content Items (which follow a content type structure), Site Pages are flexible pages where you:

  • Choose a template with predefined layout zones
  • Add widgets to each zone (Hero, Cards, Content Lists, etc.)
  • Configure each widget's content and appearance

Accessing Site Pages

Click Site Pages in the left sidebar to view all pages.

Creating a Site Page

  1. Click Create Site Page
  2. Enter the page details:
    • Title: Page title (shows in browser tab)
    • Path: URL path (e.g., about for /about)
    • Template: Select a page template
  3. Click Create

Page Templates

Templates define the layout zones available on your page:

Template Zones
Home Page hero, features, content, cta
Full Width main (single full-width zone)
With Sidebar main, sidebar
Multi Section Multiple customizable sections

The Page Editor

After creating a page, you'll see the page editor with your template's zones. Each zone can contain multiple widgets.

Adding Widgets

  1. Click Add Widget in any zone
  2. Select a widget type
  3. Configure the widget settings
  4. Click Save

See Working with Widgets for details on each widget type.

Managing Widgets

Edit

Click the Edit (pencil) icon on any widget to modify its settings.

Reorder

Drag widgets up or down within a zone to reorder them.

Delete

Click the Delete (trash) icon to remove a widget.

Draft and Publish

Site Pages use a draft/publish workflow:

  • Save as Draft: Save changes without publishing
  • Publish: Make changes live on your website
  • Discard Draft: Revert to the last published version

Page Settings

Click Settings in the page submenu to:

  • Change the page title
  • Update the URL path
  • Switch to a different template

Setting as Homepage

To use a Site Page as your homepage:

  1. Go to Settings > Configuration
  2. Under Home Page, select Site Page
  3. Choose your page from the dropdown
  4. Click Save

Revisions

Click Revisions to view and restore previous published versions of your page.

Deleting a Page

  1. Open the Site Page
  2. Click Delete
  3. Confirm the deletion
Note: If the page is set as your homepage, you'll need to change your homepage setting first.
