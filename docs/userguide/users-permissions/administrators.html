Administrators

Manage the users who have access to the Raytha admin panel. Control what each administrator can view and modify.

Accessing Administrators

Navigate to Settings > Administrators.

Creating an Administrator

  1. Click Create Administrator
  2. Enter:
    • First Name and Last Name
    • Email Address: Used for login
    • Roles: Select one or more roles
  3. Click Create

The new administrator receives a welcome email with a link to set their password.

Administrator Roles

Roles determine what an administrator can access:

Role Permissions
Super Admin Full access to everything
Admin Full access to everything
Editor Manage content and site pages

You can also create custom roles. See Roles & Permissions.

Editing an Administrator

  1. Click on the administrator's name
  2. Modify their details or roles
  3. Click Save

Resetting Password

To reset an administrator's password:

  1. Open the administrator's profile
  2. Click Reset Password
  3. They'll receive an email to set a new password

API Keys

Administrators can have API keys for programmatic access:

  1. Open the administrator's profile
  2. Click API Keys
  3. Click Generate New Key
  4. Copy the key (it's only shown once)

Suspending an Administrator

To temporarily disable access without deleting:

  1. Open the administrator's profile
  2. Click Suspend

Suspended administrators cannot log in but their account remains in the system. Click Restore to reactivate.

Deleting an Administrator

  1. Open the administrator's profile
  2. Click Remove Access
  3. Confirm the action
Note: You cannot delete your own account or the last super admin.

Your Profile

Each administrator can update their own profile:

  1. Click your name in the top-right corner
  2. Select Edit Profile or Change Password
