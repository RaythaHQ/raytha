Public Users

Manage user accounts for your public website. Create user groups and control access to protected content.

What are Public Users?

Public users are separate from administrators. They:

  • Can register on your public website
  • Log in to access protected content
  • Have profiles managed through the admin panel
  • Cannot access the admin panel

Accessing Users

Navigate to Users in the left sidebar.

Creating a User

  1. Click Create User
  2. Enter:
    • First Name and Last Name
    • Email Address
    • User Groups: Optional group membership
  3. Click Create

The user receives a welcome email to set their password.

User Registration

If enabled, users can self-register on your website. Configure registration in Settings > Authentication Schemes.

Editing a User

  1. Click on the user's name
  2. Modify their details or groups
  3. Click Save

User Status

  • Active: Can log in normally
  • Inactive: Cannot log in
  • Email Verified: Has confirmed their email address

Resetting User Password

  1. Open the user's profile
  2. Click Reset Password
  3. They'll receive an email to set a new password

User Groups

Organize users into groups for easier management and access control.

Creating a User Group

  1. Go to Users > User Groups
  2. Click Create User Group
  3. Enter:
    • Label: Display name
    • Developer Name: Identifier for templates
  4. Click Create

Assigning Users to Groups

  1. Edit the user
  2. Select groups from the User Groups list
  3. Click Save

Using Groups in Templates

Check group membership in your Liquid templates:

{% if CurrentUser.IsAuthenticated %}
  {% if CurrentUser.UserGroups contains 'premium_members' %}
    <!-- Premium content -->
  {% endif %}
{% endif %}

Deactivating a User

  1. Open the user's profile
  2. Click Deactivate

Deactivated users cannot log in. Click Activate to restore access.

Deleting a User

  1. Open the user's profile
  2. Click Delete
  3. Confirm deletion
Warning: Deleting a user is permanent and removes all their data.

Searching Users

Use the search box to find users by name or email. Filter by status or user group.
