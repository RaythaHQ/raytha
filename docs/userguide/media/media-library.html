Media Library

Upload, organize, and manage all your media files including images, documents, videos, and more.

Accessing the Media Library

Click Media in the left sidebar.

Uploading Files

Direct Upload

  1. Click Upload
  2. Select files from your computer or drag and drop
  3. Files upload automatically

From Content Editor

When editing content items, click attachment fields to upload files directly. They're automatically added to the media library.

Supported File Types

  • Images: JPG, PNG, GIF, WebP, SVG
  • Documents: PDF, DOC, DOCX, XLS, XLSX, PPT
  • Media: MP4, MP3, WAV
  • Archives: ZIP
  • Other: Depends on server configuration

Browsing Files

The media library displays files in a grid view with thumbnails for images.

Search

Use the search box to find files by name.

Sort

Sort by date uploaded, file name, or file size.

File Details

Click any file to view its details:

  • File Name: Original upload name
  • File Size: Size in bytes/KB/MB
  • Content Type: MIME type
  • Upload Date: When it was uploaded
  • Object Key: Internal storage identifier

Copying File URLs

To use a file in your content or templates:

  1. Click on the file
  2. Click Copy URL
  3. Paste the URL where needed

Using Media in Templates

Reference uploaded files by their object key:

<img src="{{ 'media-object-key' | raytha_attachment_url }}" alt="...">

For theme assets, they're automatically referenced in templates.

Storage Providers

Raytha supports multiple file storage backends:

  • Local: Files stored on the web server
  • Azure Blob Storage: Microsoft Azure cloud storage
  • Amazon S3: AWS cloud storage

The storage provider is configured via environment variables. Files display the same way regardless of backend.

File Size Limits

Maximum upload size depends on your server configuration. Default is typically 100MB per file.

Deleting Files

  1. Click on the file
  2. Click Delete
  3. Confirm deletion
Warning: Deleting a file breaks any content or templates referencing it. Check usage before deleting.

Theme Media

Media files can be associated with specific themes (CSS, JS, fonts). These are managed through Design > Themes > Media.
